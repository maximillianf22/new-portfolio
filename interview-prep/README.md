# Interview Preparation Materials

This folder contains materials for interview preparation. It's excluded from git to keep personal notes private.

## Contents

- `interview-prep.md` - Main interview preparation guide with common questions, answers, and tips
- `legacy-projects.md` - Reference list of older projects from previous portfolio (for use in interviews when relevant)

## Usage

1. Review `interview-prep.md` before interviews
2. Practice answers out loud
3. Use `legacy-projects.md` as reference when discussing breadth of experience
4. Keep this folder updated with notes from interviews
